## Typography

## Styling

Colors are defined in the **OKLCH** color space — a perceptually uniform model that keeps lightness, chroma, and hue independent. Each palette is generated as a percentage of the sRGB gamut maximum, so every hue looks equally vivid at the same step.

The accent throughout the site is **Tomato**.

I use custom CSS variables for spacing, typography, and theming.

## Code

Syntax highlighting is powered by **Expressive Code** with two themes:

- **Rose Pine Dawn** — light mode
- **Kanagawa Dragon** — dark mode

Here's an example:

```typescript
interface BlogPost {
  title: string;
  description: string;
  pubDate: Date;
  slug: string;
}

function formatDate(date: Date): string {
  return date.toLocaleDateString('en-US', {
    year: 'numeric',
    month: 'long',
    day: 'numeric',
  });
}
```

And a bit of CSS to show the syntax highlighting in action:

```css
:root {
  --color-tomato-9: oklch(57% 0.13 25);
  --color-gray-11: oklch(39% 0.01 0);
}

.blog-post {
  max-width: 720px;
  margin: 0 auto;
  padding: 3rem 1.5rem;
}
```
